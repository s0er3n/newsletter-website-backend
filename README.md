# Newletter Website Backend


## Commands:
```shell
newsletter [host=localhost] [port=80] [--reload] [--dev]

ARGS:

host=0.0.0.0  - specify host
port=80       - specify port
--reload      - reload application on save
--dev         - pass "DEBUG" = True Flag to application

OTHER ARGUMENTS:
key=value    - sets environment variable 'key' to 'value'
--flag       - sets environment variabe 'flag' to True
```
